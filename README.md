## Description

This is the official REST API for the ShopScope Project.

API and Docs are available at:

- [Shopscope](https://shopscope.herokuapp.com/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
